## 🧪 خطوات الاختبار

### المرحلة 1: تفعيل Canvas
```
1. افتح المشروع في المتصفح: http://localhost:3000
2. في ChatInputBox، اضغط على زر الأدوات (Sliders Icon)
3. اختر "Canvas" من القائمة
4. ✅ يجب أن تنقسم الشاشة ويظهر Canvas على اليمين
```

### المرحلة 2: اختيار النموذج
```
1. في Canvas Toolbar (أعلى Canvas)
2. اضغط على "Select Model" (Sparkles Icon)
3. اختر نموذج AI (مثلاً: Gemini 2.0 Flash)
4. ✅ يجب أن يظهر اسم النموذج في الـ Toolbar
```

### المرحلة 3: كتابة كود
```
1. اكتب رسالة: "Create a React button component"
2. أرسل الرسالة
3. ✅ يجب أن يبدأ AI بالكتابة في Canvas
4. ✅ يجب أن تظهر علامات تبويب الملفات
```

### المرحلة 4: تحرير تفاعلي
```
1. حدد جزء من الكود في Monaco Editor
2. Right-click ← "Ask AI to edit this"
3. اكتب طلب التعديل (مثلاً: "Add TypeScript types")
4. ✅ يجب أن يعدل AI الجزء المحدد فقط
5. ✅ يجب أن ترى البث المباشر للتعديلات
```

### المرحلة 5: تنسيق الكود
```
1. اضغط على زر "Format" في Toolbar
2. ✅ يجب أن يتم تنسيق الكود بواسطة Prettier
```

### المرحلة 6: معاينة
```
1. إذا كان الكود HTML/JS/React، اضغط على تبويب "Preview"
2. ✅ يجب أن ترى معاينة مباشرة
3. اضغط على Console Icon
4. ✅ يجب أن ترى console.log outputs
```

### المرحلة 7: مقارنة الإصدارات
```
1. بعد إجراء تعديلات، اضغط على تبويب "Changes"
2. ✅ يجب أن ترى الفروقات بين الإصدار السابق والحالي
3. جرب تبديل بين "Split View" و "Unified View"
```

### المرحلة 8: تصدير
```
1. اضغط على زر "Download" في Canvas Header
2. جرب تصدير:
   - ✅ Markdown
   - ✅ PDF
   - ✅ PNG
   - ✅ Code File
3. تحقق من جودة الملفات المحملة
```

### المرحلة 9: قوالب
```
1. في Canvas، اضغط على "Templates" (إذا كان موجود في Header)
2. أو افتح Canvas فارغ واختر قالب
3. اضغط على "Use Template"
4. ✅ يجب أن يتم ملء Canvas بكود القالب
```

### المرحلة 10: ملفات متعددة
```
1. في Toolbar، اضغط على زر "+" لإضافة ملف جديد
2. اكتب اسم الملف: "utils.ts"
3. ✅ يجب أن تظهر علامة تبويب جديدة
4. اكتب كود في الملف الجديد
5. بدل بين الملفات بالضغط على التبويبات
6. ✅ كل ملف يجب أن يحتفظ بمحتواه
```
