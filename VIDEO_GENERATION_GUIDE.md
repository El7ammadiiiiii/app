# 🎬 دليل توليد الفيديو - Google Veo 3.1

## نظرة عامة | Overview

تم دمج نظام توليد الفيديو بالكامل باستخدام Google Veo 3.1 في أداة المحادثة. يمكن للمستخدمين الآن إنشاء مقاطع فيديو احترافية بجودة 4K مع حوار وصوت ومؤثرات سينمائية.

Complete video generation system integrated using Google Veo 3.1. Users can now create professional 4K videos with dialogue, sound effects, and cinematic quality.

---

## 🚀 كيفية الاستخدام | How to Use

### الوصول إلى الأداة | Access the Tool

1. **افتح واجهة المحادثة** | Open chat interface
2. **انقر على زر الأدوات** (أيقونة Sliders) | Click the Tools button (Sliders icon)
3. **اختر "Generate Video"** من القائمة | Select "Generate Video" from menu

### أنواع التوليد | Generation Types

#### 1️⃣ نص إلى فيديو | Text to Video
- أدخل وصف نصي للفيديو المطلوب
- الأداة تولد الفيديو مباشرة من النص
- **مثال**: "سيارة رياضية حمراء تسير في طريق جبلي عند الغروب"

#### 2️⃣ صورة إلى فيديو | Image to Video
- ارفع صورة
- أضف وصف لكيفية تحريك الصورة
- الأداة تحول الصورة إلى فيديو متحرك
- **مثال صورة**: صورة منظر طبيعي + "الكاميرا تتحرك ببطء من اليسار لليمين"

#### 3️⃣ مع صور مرجعية | With Reference Images
- ارفع عدة صور مرجعية (لتحديد النمط/الأسلوب)
- أضف وصف المشهد المطلوب
- الأداة تستخدم الصور كمرجع للأسلوب الفني
- **مثال**: صور أنمي + "قط يلعب بالكرة"

#### 4️⃣ الاستكمال بين إطارين | Frame Interpolation
- ارفع إطار البداية (صورة)
- ارفع إطار النهاية (صورة)
- أضف وصف الحركة المطلوبة
- الأداة تملأ الحركة بين الإطارين
- **مثال**: صورة قبل + صورة بعد + "انتقال سلس بين المشهدين"

#### 5️⃣ تمديد الفيديو | Video Extension
- ارفع فيديو موجود
- حدد مدة التمديد (2-8 ثانية)
- أضف وصف لكيفية الاستمرار
- الأداة تمدد الفيديو بمحتوى جديد
- **مثال**: فيديو لشخص يمشي + "يستمر في المشي نحو البحر"

## ⚙️ الإعدادات | Settings

### نسبة العرض | Aspect Ratio
- **16:9** - أفقي (YouTube, TV)
- **9:16** - عمودي (TikTok, Stories)
- **1:1** - مربع (Instagram)

### الدقة | Resolution
- **4K** - أعلى جودة (3840x2160)
- **1080p** - عالي الدقة (1920x1080)
- **720p** - دقة قياسية (1280x720)

### المدة | Duration
- **4-16 ثانية** للفيديو الجديد
- **2-8 ثانية** للتمديد

## 📊 مراقبة التوليد | Generation Monitoring

### حالات التوليد | Generation States

- **⏳ Pending** - في قائمة الانتظار
- **🔄 Processing** - قيد المعالجة (يستغرق 1-3 دقائق)
- **✅ Completed** - اكتمل بنجاح
- **❌ Failed** - فشل (يظهر رسالة الخطأ)

### التحديث التلقائي
- الأداة تفحص الحالة كل 5 ثوانٍ
- يتم التحديث تلقائياً عند الاكتمال
- إشعار عند جاهزية الفيديو

## 🎯 أمثلة على النصوص الفعالة | Effective Prompts

### نصائح لكتابة نص جيد | Tips for Good Prompts

#### ✅ جيد | Good
```
كاميرا سينمائية تتحرك ببطء حول سيارة رياضية حمراء لامعة 
موقوفة في شارع مدينة مستقبلية في الليل، أضواء نيون زرقاء 
وبنفسجية تنعكس على جسم السيارة، مطر خفيف يسقط
```

#### ❌ ضعيف | Weak
```

```

### عناصر النص الفعال | Effective Prompt Elements

1. **الموضوع الرئيسي** | Main Subject
   - "امرأة شابة ترتدي فستان أحمر"

2. **الحركة/الإجراء** | Action/Movement
   - "تمشي ببطء نحو الكاميرا"

3. **البيئة/الخلفية** | Environment/Background
   - "في حديقة خضراء مليئة بالزهور"

4. **الإضاءة/الجو** | Lighting/Atmosphere
   - "في ضوء الشمس الذهبي عند الغروب"

5. **حركة الكاميرا** | Camera Movement
   - "مع تتبع سلس من الخلف"

6. **التفاصيل الإضافية** | Additional Details
   - "شعرها يتحرك مع النسيم الخفيف"

## 🔧 البنية التقنية | Technical Architecture

### الملفات | Files Created

```
src/
├── services/
│   └── veoService.ts           # Veo 3.1 API integration
├── store/
│   └── veoStore.ts             # State management with Zustand
├── components/
│   ├── chat/
│   │   └── ChatInputBox.tsx    # Updated with Video button
│   └── tools/
│       └── VideoGeneratorTool.tsx  # Main UI component
└── app/api/veo/
    ├── generate/
    │   └── route.ts            # Generation endpoint
    └── status/[operationName]/
        └── route.ts            # Status check endpoint
```

### التدفق | Flow

1. **المستخدم يضغط زر Generate Video**
2. **يملأ النموذج بالنص والإعدادات**
3. **يرفع الملفات (إن وجدت)**
4. **الضغط على "Generate Video"**
5. **API call إلى `/api/veo/generate`**
6. **الخادم يرسل طلب إلى Google Veo API**
7. **يُرجع اسم العملية (operation name)**
8. **الأداة تفحص الحالة كل 5 ثوانٍ**
9. **عند الاكتمال، يظهر الفيديو للمعاينة**
10. **يمكن تحميل الفيديو**

## 🐛 استكشاف الأخطاء | Troubleshooting

### مشكلة: الفيديو لا يكتمل
- **الحل**: انتظر حتى 3 دقائق
- تحقق من اتصال الإنترنت
- تحقق من صحة مفتاح API

### مشكلة: خطأ في الرفع
- **الحل**: تأكد من حجم الصورة/الفيديو < 10MB
- استخدم صيغ مدعومة (JPG, PNG, MP4)

### مشكلة: جودة منخفضة
- **الحل**: استخدم دقة 4K
- اكتب نص أكثر تفصيلاً
- أضف تفاصيل عن الإضاءة والحركة
