# نظام الثيمات (Theme System)

## نظرة عامة
تم تبسيط نظام الثيمات ليدعم وضعين فقط: الداكن (Dark) والفاتح (Light).

## الثيمات المتوفرة (2 Themes)

### 1️⃣ **Dark** (الوضع الداكن - الافتراضي)
- **الخلفية**: تدرج أسود مزرق `#0a0e12 → #0f1419 → #1a1f2e`
- **الاستخدام**: الوضع الرئيسي للتطبيق، مريح للعين في الإضاءة المنخفضة.
- **الطابع**: احترافي، حديث، مع تباين عالي للنصوص.

### 2️⃣ **Light** (الوضع الفاتح)
- **الخلفية**: تدرج أبيض/رمادي فاتح `#f8fafc → #ffffff → #f1f5f9`
- **الاستخدام**: للعمل في الإضاءة الساطعة أو لمن يفضل الخلفيات الفاتحة.
- **الطابع**: نظيف، واضح، وبسيط.

## الكلاسات المتاحة

### ✨ `.theme-bg`
**الاستخدام**: الخلفية الرئيسية للصفحة
```tsx
<div className="theme-bg">
  // المحتوى
</div>
```

### 🃏 `.theme-card`
**الاستخدام**: البطاقات والعناصر البارزة
```tsx
<div className="theme-card rounded-lg p-4">
  // محتوى البطاقة
</div>
```

### 📁 `.theme-sidebar`
**الاستخدام**: الـ sidebars (يمين ويسار)
```tsx
<aside className="theme-sidebar">
  // محتوى الـ sidebar
</aside>
```

### 🎯 `.theme-header`
**الاستخدام**: العناوين والـ headers
```tsx
<header className="theme-header">
  // محتوى الـ header
</header>
```

### 📋 `.theme-dropdown`
**الاستخدام**: القوائم المنسدلة والـ modals
```tsx
<div className="theme-dropdown">
  // محتوى القائمة
</div>
```

### 🎨 `.theme-surface`
**الاستخدام**: الأسطح الزجاجية والعناصر الشفافة
```tsx
<div className="theme-surface backdrop-blur-sm">
  // محتوى شفاف
</div>
```

---

## كيفية التبديل
يتم التبديل تلقائياً باستخدام `next-themes` أو عبر زر التبديل في الإعدادات.
```tsx
const { setTheme } = useTheme();
// للتبديل إلى الداكن
setTheme('dark');
// للتبديل إلى الفاتح
setTheme('light');
```
